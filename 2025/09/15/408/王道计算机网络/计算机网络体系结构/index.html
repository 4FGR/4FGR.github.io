1.1 и®Ўз®—жңәзҪ‘з»ңжҰӮиҝ°

и®Ўз®—жңәзҪ‘з»ңзҡ„жҰӮеҝө

вҖӢ и®Ўз®—жңәзҪ‘и·ҜжҳҜдёҖдёӘе°Ҷдј—еӨҡеҲҶж•Јзҡ„гҖҒиҮӘжҙҪзҡ„и®Ўз®—жңәзі»з»ҹпјҢйҖҡиҝҮйҖҡдҝЎи®ҫеӨҮдёҺзәҝи·ҜиҝһжҺҘиө·жқҘпјҢжңүеҠҹиғҪе®Ңе–„зҡ„иҪҜд»¶е®һзҺ°иө„жәҗе…ұдә«е’ҢдҝЎжҒҜдј йҖ’зҡ„зі»з»ҹгҖӮ

вҖӢ и®Ўз®—жңәзҪ‘з»ң(з®Җз§° зҪ‘з»ң)пјҡз”ұиӢҘе№Із»“зӮ№е’Ңй“ҫжҺҘиҝҷдәӣз»“зӮ№зҡ„й“ҫи·Ҝ(link)з»„жҲҗгҖӮз»“зӮ№еҸҜд»ҘжҳҜи®Ўз®—жңәгҖҒйӣҶзәҝеҷЁгҖҒдәӨжҚўжңәгҖҒи·Ҝз”ұеҷЁзӯүгҖӮ

вҖӢ жҠҠдёӨдёӘжҲ–еӨҡдёӘи®Ўз®—жңәзҪ‘з»ңдә’зӣёиҝһжҺҘиө·жқҘпјҢеҪўжҲҗи§„жЁЎжӣҙеӨ§зҡ„и®Ўз®—жңәзҪ‘з»ңпјҢд№ҹеҸҜз§°вҖңдә’иҝһзҪ‘вҖңгҖӮ

вҖӢ дә’иҒ”зҪ‘(еӣ зү№зҪ‘)жҢҮеҪ“еүҚе…ЁзҗғжңҖеӨ§зҡ„гҖҒејҖж”ҫзҡ„гҖҒз”ұдј—еӨҡзҪ‘з»ңе’Ңи·Ҝз”ұеҷЁдә’иҝһиҖҢжҲҗзҡ„зү№е®ҡи®Ўз®—жңәзҪ‘з»ңпјҢе®ғйҮҮз”ЁдәҶTCP/IPеҚҸи®®ж—ҸдҪңдёәйҖҡдҝЎи§„еҲҷгҖӮ

жіЁж„Ҹпјҡе®¶з”Ёи·Ҝз”ұеҷЁеҫҖеҫҖжҳҜи·Ҝз”ұеҷЁ+дәӨжҚўжңә+е…¶д»–еҠҹиғҪ

и®Ўз®—жңәзҪ‘з»ңзҡ„з»„жҲҗ

вҖӢ д»Һз»„жҲҗйғЁеҲҶзңӢпјҢи®Ўз®—жңәзҪ‘з»ңдё»иҰҒз”ұзЎ¬д»¶гҖҒиҪҜд»¶гҖҒеҚҸи®®дёүеӨ§йғЁеҲҶз»„жҲҗгҖӮ

вҖӢ д»Һе·ҘдҪңж–№ејҸзңӢпјҢи®Ўз®—жңәзҪ‘з»ң(иҝҷйҮҢдё»иҰҒжҢҮInternetпјҢдә’иҒ”зҪ‘) еҸҜеҲҶдёәиҫ№зјҳйғЁеҲҶе’Ңж ёеҝғйғЁеҲҶгҖӮиҫ№зјҳйғЁеҲҶз”ұдә’иҒ”зҪ‘дёҠзҡ„дҫӣз”ЁжҲ·зӣҙжҺҘдҪҝз”Ёзҡ„дё»жңәз»„жҲҗпјҢз”ЁжқҘиҝӣиЎҢйҖҡдҝЎе’Ңиө„жәҗе…ұдә«;ж ёеҝғйғЁеҲҶз”ұеӨ§йҮҸзҪ‘з»ңе’ҢиҝһжҺҘиҝҷдәӣзҪ‘з»ңзҡ„и·Ҝз”ұеҷЁз»„жҲҗпјҢе®ғдёәиҫ№зјҳйғЁеҲҶжҸҗдҫӣиҝһйҖҡжҖ§е’ҢдәӨжҚўжңҚеҠЎгҖӮ

вҖӢ д»ҺеҠҹиғҪз»„жҲҗдёҠзңӢпјҢи®Ўз®—жңәзҪ‘и·Ҝз”ұйҖҡдҝЎеӯҗзҪ‘е’Ңиө„жәҗеӯҗзҪ‘з»„жҲҗгҖӮ

  • жҸҗй«ҳеҸҜйқ жҖ§

вҖӢ з”өи·ҜдәӨжҚў(Circuit Switching): йҖҡиҝҮзү©зҗҶзәҝи·Ҝзҡ„иҝһжҺҘпјҢеҠЁжҖҒең°еҲҶй…Қдј иҫ“и·Ҝзәҝзҡ„иө„жәҗгҖӮ

  • еҠ йҖҹдј иҫ“

  • е№ҝж’ӯејҸзҪ‘з»ң

вҖӢ дёҖеҸ°и®Ўз®—жңәеҸ‘йҖҒж•°жҚ®еҲҶз»„ж—¶пјҢе№ҝж’ӯиҢғеӣҙеҶ…жүҖжңүи®Ўз®—жңәйғҪдјҡж”¶еҲ°иҜҘеҲҶз»„пјҢе№¶йҖҡиҝҮжЈҖжҹҘеҲҶз»„зҡ„зӣ®зҡ„ең°еқҖеҶіе®ҡжҳҜеҗҰжҺҘж”¶

  • зӮ№еҜ№зӮ№зҪ‘з»ң

вҖӢ ж•°жҚ®еҸӘдјҡд»ҺеҸ‘йҖҒж–№вҖқзӮ№еҜ№зӮ№вҖңеҸ‘еҲ°жҺҘж”¶ж–№пјҢзІҫеҮҶйҖҒиҫҫгҖӮ

  • зҺҜеҪўз»“жһ„вҖ”вҖ”йҖҡиҝҮвҖңд»ӨзүҢвҖқи§ЈеҶіжҖ»зәҝй—®йўҳ
  • жҳҹеҪўз»“жһ„вҖ”вҖ”з”ұдёӯеӨ®и®ҫеӨҮе®һзҺ°ж•°жҚ®зҡ„зӮ№еҜ№зӮ№дј иҫ“
  • зҪ‘зҠ¶з»“жһ„вҖ”вҖ”зӮ№еҜ№зӮ№дј иҫ“пјҢзҒөжҙ»гҖҒеҸҜйқ жҖ§й«ҳ

дҝЎйҒ“(Channel) иЎЁзӨәжҹҗдёҖж–№еҗ‘дј йҖҒдҝЎжҒҜзҡ„йҖҡйҒ“пјҢдҪҶе№¶йқһйҖҡдҝЎзәҝи·ҜгҖӮ

  • йҖҹзҺҮпјҢеҚ•дҪҚжңүMbpsпјҢMB/Sзӯү

  • еёҰе®ҪпјҢжҹҗдҝЎйҒ“жүҖиғҪдј йҖҒзҡ„жңҖй«ҳж•°жҚ®зҺҮ

  • еҗһеҗҗйҮҸпјҢжҢҮеҚ•дҪҚж—¶й—ҙеҶ…йҖҡиҝҮжҹҗдёӘзҪ‘з»ң(жҲ–дҝЎйҒ“гҖҒжҺҘеҸЈ)зҡ„е®һйҷ…ж•°жҚ®йҮҸ пјҢеҸ—еёҰе®ҪгҖҒеӨҚжқӮзҡ„зҪ‘з»ңиҙҹиҪҪжғ…еҶөеҪұе“ҚгҖӮ

  • ж—¶е»¶пјҢж•°жҚ®д»ҺзҪ‘з»ң(жҲ–й“ҫи·Ҝ)зҡ„дёҖз«Ҝдј йҖҒеҲ°еҸҰдёҖз«ҜжүҖйңҖзҡ„ж—¶й—ҙгҖӮ

    • жҖ»ж—¶е»¶ = еҸ‘йҖҒж—¶е»¶ + дј ж’ӯж—¶е»¶ + еӨ„зҗҶж—¶е»¶ + жҺ’йҳҹж—¶е»¶
    • еҸ‘йҖҒж—¶е»¶(дј иҫ“ж—¶е»¶)пјҢиҠӮзӮ№е°Ҷж•°жҚ®жҺЁеҗ‘дҝЎйҒ“жүҖеҢ–зҡ„ж—¶й—ҙгҖӮ
    • дј ж’ӯж—¶е»¶пјҢз”өзЈҒжіўеңЁдҝЎйҒ“дёӯдј ж’ӯдёҖе®ҡи·қзҰ»жүҖиҠұзҡ„ж—¶й—ҙгҖӮ
    • еӨ„зҗҶж—¶е»¶пјҢиў«и·Ҝз”ұеҷЁеӨ„зҗҶжүҖиҠұзҡ„ж—¶й—ҙгҖӮ
    • жҺ’йҳҹж—¶е»¶пјҢж•°жҚ®жҺ’йҳҹиҝӣе…ҘпјҢжҺ’йҳҹеҸ‘еҮәиҝҮзЁӢдёӯи·Ҝз”ұеҷЁжүҖиҠұзҡ„ж—¶й—ҙгҖӮ

  • ж—¶е»¶еёҰе®Ҫз§ҜпјҢдј ж’ӯж—¶е»¶ \(\times\) еёҰе®ҪпјҢдёҖжқЎй“ҫи·ҜдёӯпјҢе·Ід»ҺеҸ‘йҖҒз«ҜеҸ‘еҮәдҪҶе°ҡжңӘеҲ°иҫҫжҺҘж”¶з«Ҝзҡ„жңҖеӨ§жҜ”зү№ж•°гҖӮ

  • еҫҖиҝ”ж—¶е»¶(RTT)пјҡиЎЁзӨәд»ҺеҸ‘йҖҒж–№еҸ‘йҖҒе®Ңж•°жҚ®пјҢеҲ°еҸ‘йҖҒж–№ж”¶еҲ°жқҘиҮӘжҺҘж”¶ж–№зҡ„зЎ®и®ӨжҖ»е…ұз»ҸеҺҶзҡ„ж—¶й—ҙгҖӮ

  • дҝЎйҒ“еҲ©з”ЁзҺҮ: жҹҗдёӘдҝЎйҒ“жңүзҷҫеҲҶд№ӢеӨҡе°‘зҡ„ж—¶й—ҙжҳҜжңүж•°жҚ®йҖҡиҝҮзҡ„гҖӮ(еӨӘдҪҺжөӘиҙ№иө„жәҗпјҢеӨӘй«ҳе®№жҳ“зҪ‘з»ңжӢҘеЎһ)

    • \(\frac{жңүж•°жҚ®йҖҡиҝҮзҡ„ж—¶й—ҙ}{жңүж•°жҚ®йҖҡиҝҮзҡ„ж—¶й—ҙ+жІЎжңүж•°жҚ®йҖҡиҝҮзҡ„ж—¶й—ҙ}\)

1.2 и®Ўз®—жңәзҪ‘з»ңдҪ“зі»з»“жһ„дёҺеҸӮиҖғжЁЎеһӢ

и®Ўз®—жңәзҪ‘з»ңеҲҶеұӮз»“жһ„

е®һдҪ“пјҡеңЁи®Ўз®—жңәзҪ‘з»ңзҡ„еҲҶеұӮз»“жһ„дёӯпјҢз¬¬ n еұӮдёӯзҡ„жҙ»еҠЁе…ғзҙ пјҲиҪҜд»¶+зЎ¬д»¶пјүйҖҡеёёз§°дёәз¬¬ n еұӮе®һдҪ“гҖӮдёҚеҗҢжңәеҷЁдёҠзҡ„еҗҢдёҖеұӮз§°дёәеҜ№зӯүеұӮпјҢеҗҢдёҖеұӮзҡ„е®һдҪ“з§°дёәеҜ№зӯүе®һдҪ“гҖӮ

еҚҸи®®пјҡеҚізҪ‘з»ңеҚҸи®®пјҢжҳҜжҺ§еҲ¶еҜ№зӯүе®һдҪ“д№Ӣй—ҙиҝӣиЎҢйҖҡдҝЎзҡ„и§„еҲҷзҡ„йӣҶеҗҲпјҢжҳҜж°ҙе№ізҡ„гҖӮ

жҺҘеҸЈпјҡеҗҢдёҖиҠӮзӮ№еҶ…зӣёйӮ»дёӨеұӮзҡ„е®һдҪ“зҡ„йҖ»иҫ‘жҺҘеҸЈпјҢеҸҲз§°дёәжңҚеҠЎи®ҝй—®зӮ№гҖӮ

жңҚеҠЎпјҡжҢҮдёӢеұӮдёәзҙ§йӮ»зҡ„дёҠеұӮжҸҗдҫӣзҡ„еҠҹиғҪи°ғз”ЁпјҢе®ғжҳҜеһӮзӣҙзҡ„гҖӮ

YSCSеҚҸи®®пјҲеҺӢзј©дј иҫ“еҚҸи®®пјү

йҰ–йғЁжҢҮжҳҺеҺӢзј©з®—жі•пјҢж №жҚ®йҰ–йғЁдҝЎжҒҜи§ЈеҺӢж•°жҚ®гҖӮ

еҚҸи®®ж•°жҚ®еҚ•е…ғпјҡеҜ№зӯүеұӮж¬Ўд№Ӣй—ҙдј йҖҒзҡ„ж•°жҚ®еҚ•дҪҚгҖӮз¬¬ n еұӮзҡ„ PDU и®°дёә n-PDU

жңҚеҠЎж•°жҚ®еҚ•е…ғпјҡдёәе®ҢжҲҗдёҠдёҖеұӮе®һдҪ“жүҖиҰҒжұӮзҡ„еҠҹиғҪиҖҢдј йҖҒзҡ„ж•°жҚ®гҖӮз¬¬ n еұӮзҡ„ SDU и®°дёә n-SDU

еҚҸи®®жҺ§еҲ¶дҝЎжҒҜпјҡжҺ§еҲ¶еҚҸи®®ж“ҚдҪңзҡ„дҝЎжҒҜгҖӮз¬¬ n еұӮзҡ„ PCL и®°дёә n-PCL

дёүеұӮзҡ„е…ізі»дёәпјҡ \(n-SDU + n-PCL= n-PDU = (n-1)-SDU\)

еҚҸи®®дёүиҰҒзҙ 

  • иҜӯжі•пјҡж•°жҚ®дёҺжҺ§еҲ¶дҝЎжҒҜзҡ„ж јејҸ
  • иҜӯд№үпјҡеҚійңҖиҰҒеҸ‘еҮәдҪ•з§ҚжҺ§еҲ¶дҝЎжҒҜгҖҒе®ҢжҲҗдҪ•з§ҚеҠЁдҪңеҸҠдҪңеҮәдҪ•з§Қеә”зӯ”гҖӮ
  • еҗҢжӯҘ(ж—¶еәҸ)пјҡжү§иЎҢеҗ„з§Қж“ҚдҪңзҡ„жқЎд»¶гҖҒж—¶и®ёе…ізі»зӯүгҖӮ
